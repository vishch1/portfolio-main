## 🚀 About This Project

This is my personal portfolio website where I showcase my projects, technical skills, and resume. 

I built this project by customizing a template originally created by [Soumyajit Behera](https://github.com/soumyajit4419/Portfolio).

## 🛠 Built With

- **React.js**
- **Node.js**
- **CSS3 / React-Bootstrap**
- **Vercel** (Hosting)

## ✨ Features

* **📖 Multi-Page Layout:** Home, About, Projects, and Resume sections.
* **📱 Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
* **🎨 Easy Customization:** Built with modular React components.
